NTP Clock Firmware Installer

Install firmware and configure your ESP32-S3 NTP Clock

⚠️ Browser Compatibility This installer requires Chrome, Edge, or another Chromium-based browser with Web Serial API support.
WiFi Configuration
Timezone Configuration
Set to 3600 (1 hour) for US timezones with DST. The firmware will automatically apply this offset during daylight saving periods. Set to 0 if your timezone doesn't observe DST.
Installation Steps:
1. Connect your ESP32-S3 board via USB
2. Fill in WiFi credentials and timezone above
3. Click the install button below
4. Select your ESP32 device when prompted
5. Wait for installation to complete
Alternative: AP Mode Configuration
If you prefer not to use the web installer, you can flash firmware manually and configure via AP mode. The device will create a WiFi access point "NTP-Clock-AP" that you can connect to and configure at 192.168.4.1
